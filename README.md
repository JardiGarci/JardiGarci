## Hi, I'm Jardi García 👋

M.Sc. student in Advanced Technology at IPN, Mexico City.  
I build machine learning pipelines for medical image analysis, combining **multifractal analysis** and **topological data analysis** to extract interpretable features from complex data.

---

### 🔬 What I work on

- End-to-end ML pipelines for image classification (chest X-rays, mammograms)
- Feature engineering with MF-DFA 2D and persistent entropy
- Supervised learning with interpretability focus (RF, SVM, MLP, k-NN)
- Deep learning for segmentation (U-Net / TensorFlow)
